Description détaillée

Protocol header for glXSetClientInfo2ARB

The glXSetClientInfo2ARB protocol differs from glXSetClientInfoARB in that the list of OpenGL versions supported by the client is 3 CARD32 values per version: major version, minor version, and supported profile mask.

This protocol replaces GLXClientInfo and GLXSetClientInfoARB.
