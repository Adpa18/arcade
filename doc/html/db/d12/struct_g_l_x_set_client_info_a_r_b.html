Description détaillée

Protocol header for glXSetClientInfoARB

This structure is follwed by numVersions * 2 CARD32 values listing the OpenGL versions supported by the client. The pairs of values are an OpenGL major version followed by a minor version. For example, 

 CARD32 versions[4] = { 2, 1, 3, 0 };

says that the client supports OpenGL 2.1 and OpenGL 3.0.

These are followed by numGLExtensionBytes bytes of STRING8 containing the OpenGL extension string supported by the client and up to 3 bytes of padding.

The list of OpenGL extensions is followed by numGLXExtensionBytes bytes of STRING8 containing the GLX extension string supported by the client and up to 3 bytes of padding.

This protocol replaces GLXClientInfo.
